# Diplomski rad

Diplomski rad za FER

## Getting Started

### Dependencies

Python3

- uvicorn
- fastapi
- numpy
- tk
- requests
- pandas
- sklearn
- pydantic

```
pip install -r requirements.txt
```

### Executing program

- Otvoriti dvije konzole
- U prvoj konzoli pokrenuti API

```
python REST_API.py
```

- U drugoj pokrenuti primjer za API

```
python main_api.py
```

- GUI verzija

```
python main_gui.py
```
